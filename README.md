Projet basé sur Astro avec composants React et outillage de tests.

## Fonctionnalités

- Rendu performant orienté contenu
- Composants React intégrables
- Stylisation via Tailwind CSS
- Socle de tests unitaires et e2e

## Stack technique

- Astro
- React
- TypeScript
- Tailwind CSS
- Jest / Playwright

## Prérequis

- Node.js LTS
- npm

## Démarrage rapide

```bash
npm install
npm run dev
```

## Commandes utiles

```bash
npm run dev
npm run build
npm run test
```
